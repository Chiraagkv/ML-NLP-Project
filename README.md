# Statistical Machine Learning Project
## Text Classification using Classical ML: Sparse Features vs Contextual Embeddings

- This project explores Natural Language Processing (NLP) in the pre-deep learning paradigm by comparing classical machine learning models across two types of feature representations:

    - Sparse features (Bag-of-Words, TF-IDF)
    - Contextual embeddings (from pretrained transformer encoders)

* The focus is on understanding how different feature representations impact model performance, rather than building end-to-end deep learning systems.
